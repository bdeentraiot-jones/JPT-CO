# JPT & Co — Website

## 📁 File Structure
```
jpt-website/
├── index.html          ← Main website (all 5 pages)
├── css/
│   └── style.css       ← All styles
├── js/
│   └── main.js         ← Animations, routing, interactions
└── README.md           ← This file
```

## 🚀 How to Run
Simply open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).
No server required — it works as a static website.

## 📄 5 Pages Included
1. **Home** — Animated hero, company story timeline, why-us features
2. **Services** — 9 service cards with colourful headers and tags
3. **Branches** — 6 branch cards (1 HQ + 5 branches) with map embed
4. **Gallery** — Filterable photo gallery + Google Photos connect section
5. **Contact** — Enquiry form, contact cards, testimonials, map
